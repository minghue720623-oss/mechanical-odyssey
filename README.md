# 機械奧德賽 (Mechanical Odyssey)

![Python](https://img.shields.io/badge/Python-3.x-blue?style=for-the-badge&logo=python)
![Pygame](https://img.shields.io/badge/Pygame-Latest-green?style=for-the-badge)

這是一款由 **PixelForge AI Team** 開發的懷舊機械風格動作平台遊戲。玩家扮演一塊受損的晶片，在龐大的機械廢墟中透過電磁吸附與爬行機制，穿越重重關卡以求生。

## 遊戲特色
- **獨特吸附機制**：透過 `L-CTRL` 吸附於牆面與天花板，進行靈活的垂直與橫向移動。
- **資源策略**：必須管理有限的電量，透過收集數據片段來延長動作時間。
- **復古體驗**：模擬懷舊遊戲風格與機械美學，帶來純粹的動作挑戰。
- **雙關卡設計**：挑戰多樣的地形與爬行路線。

## 安裝與運行
1. 確保已安裝 Python 3.x。
2. 安裝必要的套件：
   ```bash
   pip install pygame
   ```
3. 執行遊戲：
   ```bash
   python main.py
   ```
